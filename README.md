# 📊 Business Intelligence Sales & Profit Analysis Dashboard

## 🚀 Project Overview
This project demonstrates how Business Intelligence (BI) tools can be used to analyze sales and profit data and generate actionable insights for decision-making.

The goal is to transform raw supermarket data into meaningful visual insights using interactive dashboards.

---

## 🎯 Objectives
- Analyze sales performance across product categories
- Evaluate profit distribution across regions
- Identify trends and patterns in business performance
- Build an interactive dashboard for decision support

---

## 🛠️ Tools & Technologies
- Microsoft Power BI
- Microsoft Excel
- Data Cleaning & Transformation
- Data Visualization

---

## 📂 Dataset
- Source: Kaggle (Retail Sales Dataset)
- Features:
  - Product
  - Category
  - Region
  - Order Date
  - Total Sales
  - Total Profit

---

## 📊 Dashboard Features
- KPI Cards (Total Sales, Total Profit)
- Sales Trend Analysis (Line Chart)
- Sales by Category (Bar Chart)
- Profit by Region (Pie Chart)
- Interactive Filters (Slicers)
